Descriptive Statistics: Core Concepts

  • descriptive statistics consists of methods for organizing, summarizing, and presenting data in a way that provides insight into the characteristics of the dataset

  • involves translacting raw biological or clinical data into structured summaries and visualizations that reflect the underlying variable type and probability structure of the data

  • descriptive statistics is often the first step in understanding one’s data

Before modeling, hypothesis testing, or prediction one must determine:

  • what kind of variable is being measured
  • which summaries are mathematically appropriate
  • which plots reflect the structure of the variable

Descriptive Statistics and Variable Types

In descriptive statistics first consider:

What type of variable am I describing?

  • in analysis, variables arise from different sources:

    • mutation status [present/absent] → categorical
    • tumor stage [I–IV] → ordinal
    • mutation counts → discrete
    • biomarker levels → continuous

  • each data structure determines a distinct set of statistical properties, e.x.:

  • set of allowable values the variable can take

  • how variability appears across observations

  • which numerical summaries appropriately represent the data

  • which visualizations accurately reflect the distribution

  • starting with continuing the variable being described you allow for:

    • summaries align with the data-generating process
    • visualizations reflect the true distribution of values

Variable Types Used in Descriptive Statistics

Variable types used in analysis:

  • categorical nominal variables
  • ordinal variables
  • count variables
  • continuous variables

How Probability Connects to Description

  • probability tells us how often values or outcomes occur

  • probability defined based on outcome variables:

    • categorical variables: probabilities of categories
    • ordinal variables: probabilities across ordered levels
    • count variables: probabilities of integer counts
    • continuous variables: density across a numeric range

Code Example: Inflammatory Signaling Pathway

mutation_summary <- biomed_sim |>
  count(mutation_status) |>
  mutate(prob = n / sum(n))

mutation_summary
# A tibble: 2 × 3
  mutation_status     n  prob
  <fct>           <int> <dbl>
1 Absent            115 0.719
2 Present            45 0.281

Key parts of the code

  • count(mutation_status) counts the number of patients in each category
  • mutate(prob = n / sum(n)) converts counts into proportions
  • prob column: probability distribution for mutation status in this sample

How to interpret the output

  • n gives the observed number of patients in each class
  • prob gives the estimated probability that a randomly selected patient from this sample falls into that class
  • probabilities sum to 1 because every patient must belong to exactly one mutation-status category

Categorical Example: Plot

ggplot(mutation_summary, aes(x = mutation_status, y = prob)) +
  geom_col() +
  scale_y_continuous(labels = percent_format()) +
  labs(
    title = "Mutation Status: Empirical Category Probabilities",
    x = "Mutation status",
    y = "Sample proportion"
  ) + theme_bw()

Interpretation

  • categorical variables are described by counts, proportions, and category probabilities
  • X-axis: contains discrete categories
  • bar height: represents the probability associated with each category

Continuous Example: IL-6

il6_summary <- biomed_sim |>
  summarise(
    n = n(),
    mean_il6 = mean(il6_pg_ml),
    median_il6 = median(il6_pg_ml),
    sd_il6 = sd(il6_pg_ml),
    iqr_il6 = IQR(il6_pg_ml),
    min_il6 = min(il6_pg_ml),
    max_il6 = max(il6_pg_ml)
  )

il6_summary
# A tibble: 1 × 7
      n mean_il6 median_il6 sd_il6 iqr_il6 min_il6 max_il6
  <int>    <dbl>      <dbl>  <dbl>   <dbl>   <dbl>   <dbl>
1   160     8.52       7.45   4.80    5.57    1.96    31.6

Key parts of the code

  • summarise() reduces the full vector of IL-6 values into a set of descriptive statistics
  • mean() and median(): describe summary statistics
  • sd() and IQR(): data spread - standard deviation and interquartile range
  • min() and max() : defines the observed range

Interpretation

  • if the mean > median: data is often right skewed
  • larger standard deviation : greater dispersion around the mean
  • IQR captures the width of the middle 50% of the data

Continuous Example: IL-6

Interpretation

  • IL-6 is measured on a continuous numeric scale
  • histogram groups nearby values into bins → overall distributional shape becomes visible
  • shaded region represents the interquartile range (IQR), capturing the middle 50% of observed values
    • IQR spans from the 25th percentile (Q1) to the 75th percentile (Q3)
  • dashed line marks the median, \(\rightarrow\) central location of the distribution

Summary

  • analysis begins by identifying what kind of variable is being measured in descriptive statistics

  • review for descriptive statistics:

observed values → variable type → probability structure → summary → plot observed values

  • categorical variables \(\rightarrow\) described with counts, proportions, and bar plots

  • ordinal variables \(\rightarrow\) described with ordered counts, ordered proportions, and plots that preserve rank

  • count variables \(\rightarrow\) described through frequencies of integer outcomes and probability mass functions

  • continuous variables \(\rightarrow\) described through center, spread, and distribution shape using summaries such as the mean, median, SD, and IQR

    • plots: histograms or density-based displays
